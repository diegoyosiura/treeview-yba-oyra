# Ybá-Oyrá

Ybá-Oyrá is a lightweight library designed for treeview.

## Installation

```bash
npm install yba-oyra
Usage
typescript
Copiar código
import { greet } from 'yba-oyra';

console.log(greet('World'));
// Output: Hello, World! Welcome to Ybá-Oyrá.
```
